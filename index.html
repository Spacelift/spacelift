Spacelift is a lightweight Scala framework for rapid creation of highly-scalable, message queue-backed actors, with publicly-exposed REST APIs.

Getting Started at The Speed of Light

Toss the following into your project's dependencies:

libraryDependencies += "space.spacelift" %% "spacelift" % "1.0.0-SNAPSHOT"

Define your actors and messages as normal:

case class SomeMessage()
case class SomeOtherKindOfMessage(name: String, value: String)
case class SomeResponse(responseData: String)
case class AnotherResponse(responseData: String, isEqual: Boolean)

class Foo extends Actor {
  def receive: Actor.Receive = {
    case msg: SomeMessage => {
      // handler logic goes here...
      sender ! SomeResponse("foobar")
    }
    case otherMsg: SomeOtherKindOfMessage => {
      // handler logic goes here...
      otherActorRef ! AnotherResponse("foobar", otherMsg.name == otherMsg.value)
    }
  }
}

Import Spacelift when creating your Actors to be exposed by REST, and use its extension method:

import space.spacelift.actors._

// ...

val actorSystem = ActorSystem("system")
actorSystem.restfulActorOf(Props[Foo], "foo") // Exposes actor at /foo

Roll in some configuration info in your application.conf:

spacelift {
  queue {
    provider="rabbitmq"
    secure=true
    host="some.host.com"
    port=5671
    username="username"
    password="password"
  }
}

Run your application, and make a request against your new endpoint:

$ curl -X GET -H "Accept: application/json" http://localhost:9000/foo/msg

> { "responseData" : "foobar" }
